# Slots

Slots aims to become a forecasts alerter for water sports, based on users' favorites spots.

### Basic commands

Migrate :

```
make migrate-db
```

Run the project :

```
make run-dev
```
Slots app runs on <http://0.0.0.0:8000>

Swagger UI is on <http://0.0.0.0:8000/docs>

Run tests :

```
make test
```
